BootUp

An application build with the solid foundation of Ruby on Rails and a modern user interface powered by React-Redux.
Inspired by Kickstarter, BootUp allows creators generate support for their projects before they head out to raise funding. Even with the many database resources necessary to present a flushed-out, fully featured project, querying is optimized to fetch all relevant records with a single request thanks to the power of Rails ActiveRecord.

RubyQuarry

An object-relational mapping library for managing SQLite3 databases with syntactic Ruby. Reduces the need for writing raw SQL and adds methods for fetching related resources. Additionally, RubyQuarry can reduce development time - and developer frustration - by providing a clear set of conventions in database and model setup, reducing the need for lengthy configuration.

Space Heroes

A modern, modular, browser-based re-imagining of an Atari classic. Powered by JavaScript ES6, Space Heroes make development and roll-out of new features quick and easy by leveraging the new module syntax. The power of Canvas makes possible reactive graphics, even simulated particle effects, in the browser.
